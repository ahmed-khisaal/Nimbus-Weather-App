# 🌤️ Nimbus Weather App

A modern and responsive weather application built with **HTML, CSS, and JavaScript** that provides real-time weather updates using the **Open-Meteo API**. The application allows users to search for weather information for any location worldwide and supports automatic geolocation for local weather updates.

---

## ✨ Features

* 🌍 Search weather by city or location.
* 📍 Automatic geolocation support.
* 🌡️ Real-time temperature and weather conditions.
* ☁️ Dynamic weather icons and descriptions.
* 📱 Fully responsive design for desktop and mobile devices.
* 🎨 Modern and intuitive user interface.
* ⚡ Fast and lightweight with API integration.
* 🔄 Error handling for invalid locations and network issues.

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
* **Open-Meteo Weather API**
* **Open-Meteo Geocoding API**

## 🚀 Getting Started

### Prerequisites

* A modern web browser.
* Internet connection for API requests.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Nimbus-Weather-App.git
```

2. Navigate to the project directory:

```bash
cd Nimbus-Weather-App
```

3. Open `index.html` in your preferred browser.

---

## 📖 How It Works

1. Enter the name of a city or location.
2. The application fetches geographical coordinates using the Open-Meteo Geocoding API.
3. Weather data is retrieved from the Open-Meteo Weather API.
4. Current weather information is displayed with corresponding icons and details.
5. Users can also allow location access to automatically view local weather conditions.
